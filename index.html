HTML Elements/Attribute Reference

Elements Tags

Tag Name Description
<body> Body It defines the document's body.
<head> Head It contains metadata/information for the document.
<h1> Main heading This is a heading, it emphasizes text.
<p> Paragraph It contains a block of plain text.
<a> Anchor It is used to specify the text that is the "anchor" for a link.
<button> Button This is a clickable button.
<table> Table The wrapper element for all HTML tables.

Attributes

Attribute Name Description
class Class Specifies one or more classnames for an element (refers to a class in a style sheet).
id Identifier Specifies a unique id for an element.
style Style Specifies an inline CSS style for an element.
lang Language Specifies the language of the element's content.
href Hypertext Reference Specifies the URL of the page the link goes to.
src Source Specifies the URL of the media file.
alt Alternative Text Specifies an alternate text when the original element fails to display.
target Target Specifies the target for where to open the linked document or where to submit the form.
